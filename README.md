# FLEX — Handoff Public Assets

Screenshots publics de la v1.0.19 de FLEX (yako-san/flex-rev-app),
hébergés ici pour que la session V2 (yako-san/flex) puisse les fetcher
via raw URLs sans avoir besoin d'accès au repo privé V1.

**Usage** : ces images sont une **référence visuelle read-only** pour
le port UI V1→V2 (Sprint 4 β+). Ne pas modifier sans coordination avec
la session V1.
